异或，可以理解为不进位加法，1 + 1 = 0，0 + 0 = 0，1 + 0 = 1

异或的性质

  • 交换律：可任意交换运算因子的位置，结果不变
  • 结合律：即（a ^ b）^ c == a ^ ( b ^ c )
  • 对于任何数 x，都有 x ^ x = 0, x ^ 0 = x，同自己求异或为 0，同 0 求异或为自己
  • 自反性：A ^ B ^ B = A ^ 0 = A，连续和同一个因子做异或运算，最终结果为自己
